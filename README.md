## 🚀 Sobre o Projeto  

O **Gerenciador de Tarefas** é uma aplicação web desenvolvida em:  

![Angular](https://img.shields.io/badge/Angular-DD0031?style=for-the-badge&logo=angular&logoColor=white)  
![Node.js](https://img.shields.io/badge/Node.js-339933?style=for-the-badge&logo=nodedotjs&logoColor=white)  
![MongoDB](https://img.shields.io/badge/MongoDB-47A248?style=for-the-badge&logo=mongodb&logoColor=white)  

Ele permite que os usuários **gerenciem suas tarefas diárias**, oferecendo funcionalidades como:  
✅ Criação, edição e exclusão de tarefas  
✅ Organização e definição de prioridades  
✅ Interface intuitiva e responsiva  

---

## 🎯 Funcionalidades  

- ✅ Criar, editar e excluir tarefas 📌  
- ✅ Definir prioridades 🚀  
- ✅ Interface intuitiva e responsiva 🎨  
- ✅ Persistência de dados via MongoDB 🔗
